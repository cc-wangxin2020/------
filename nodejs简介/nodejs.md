#### Node.js
* 运行在服务器端的js
* 用来编写服务器
* 特点：
  * 单线程、异步、非阻塞（不阻塞其他程序的运行）
  * 统一API

#### Nvm - 控制不同的node版本
* 命令
  * nvm list - 显示已安装的node版本
  * nvm install 版本 - 安装指定版本的node
  * 配置nvm的镜像服务器
  * nvm node_mirror https://npmmirror.com/mirrors/node/
  * nvm use 版本 - 指定要使用的node版本

#### node.js和Javascript区别：
* ECMAScript(node有) 、DOM（node没有）、 BOM（node没有）
